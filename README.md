# Software Concepts

Daniel Jackson's [software concepts](https://essenceofsoftware.com) formalized using [Alloy](https://alloytools.org) and [TLA+](https://lamport.azurewebsites.net/tla/tla.html).

The repository includes formal models of both the individual concepts and of apps built by composing those concepts using synchronization rules (aka *reactions*).
